反码  正数一样   负数 第一个符号位为1 剩下  按位取反 1111 1110 表示-1    1+ -1    二进制计算为负数  则结果为1111 1111 表示-0

补码 整数一样， 负数是反码的基础上加以 0000 0001 补码是 1111 1110 + 1 就是 1111 1111  1+ -1 就是 0 正确
